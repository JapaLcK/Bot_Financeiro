Política de Privacidade

Última atualização: 5 de abril de 2026

Esta Política de Privacidade descreve como o ChatCoin ("nós", "nosso") coleta, usa e protege as informações fornecidas pelos usuários ao interagir com nosso serviço via WhatsApp.

1. Informações que coletamos

  • Número de telefone do WhatsApp utilizado para contato
  • Mensagens enviadas ao bot para fins de processamento financeiro
  • Dados financeiros inseridos pelo usuário (transações, categorias, valores)

2. Como usamos as informações

  • Para processar e registrar transações financeiras informadas pelo usuário
  • Para gerar relatórios e resumos financeiros personalizados
  • Para enviar notificações e relatórios diários solicitados pelo usuário

3. Compartilhamento de dados

Não vendemos, alugamos ou compartilhamos suas informações pessoais com terceiros, exceto quando exigido por lei ou necessário para operação do serviço (ex: infraestrutura de hospedagem).

4. Armazenamento e segurança

Os dados são armazenados em banco de dados seguro com acesso restrito. Adotamos medidas técnicas e organizacionais para proteger suas informações contra acesso não autorizado, perda ou alteração.

5. Retenção de dados

Os dados são mantidos enquanto o usuário utilizar o serviço. Você pode solicitar a exclusão dos seus dados a qualquer momento entrando em contato conosco.

6. Seus direitos

  • Acessar os dados que armazenamos sobre você
  • Solicitar correção de dados incorretos
  • Solicitar exclusão dos seus dados

7. Contato

Para dúvidas ou solicitações relacionadas à privacidade, entre em contato pelo e-mail: lucaskuramoti06@gmail.com

8. Alterações nesta política

Podemos atualizar esta política periodicamente. Notificaremos os usuários sobre mudanças significativas através do próprio serviço.
